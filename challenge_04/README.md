# Challenge 04 - The Live Twitter Feed (Console Based)

A Java-based challenge focusing on Java basics.

## Description

Build a console-based application that simulates a live Twitter feed.

## Learning Objectives

- Data structures (lists, queues)
- String formatting and output
- Time and date handling
- Java fundamentals
